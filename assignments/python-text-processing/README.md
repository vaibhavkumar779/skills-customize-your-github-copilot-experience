# 📘 Assignment: Python Text Processing

## 🎯 Objective

Practice Python text processing by reading from files, manipulating strings, and producing formatted output.

## 📝 Tasks

### 🛠️ Read and Process Text Files

#### Description

Write a function that reads text from a file and processes its content line by line.

#### Requirements
Completed program should:

- Open a text file for reading
- Read each line from the file
- Strip whitespace and normalize the text for processing
- Return the cleaned text lines as a list

### 🛠️ Count Words and Characters

#### Description

Implement a function to analyze text and count words, lines, and characters.

#### Requirements
Completed program should:

- Count the number of lines in the text
- Count the total number of words
- Count the total number of characters
- Return the counts as a dictionary or printed summary

### 🛠️ Search and Replace Text

#### Description

Add functionality to search for a target word and replace it with a new word.

#### Requirements
Completed program should:

- Search text for a specific target word
- Replace all occurrences with a replacement word
- Return or save the modified text
- Preserve the original file structure and spacing where possible
