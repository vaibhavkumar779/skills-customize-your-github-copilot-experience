# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Use Python data visualization tools to create charts and graphs from a dataset, then customize labels and save the output.

## 📝 Tasks

### 🛠️ Load and Explore Dataset

#### Description

Read the provided dataset and prepare it for visualization.

#### Requirements
Completed program should:

- Load data from `sample-data.csv`
- Parse numeric values correctly
- Print a short summary of the dataset

### 🛠️ Create Charts

#### Description

Build at least two different visualizations that represent the dataset clearly.

#### Requirements
Completed program should:

- Create at least two different chart types (for example, line chart and bar chart)
- Use labels, titles, and legends where appropriate
- Display the charts or save them as image files

### 🛠️ Customize Visualization Output

#### Description

Enhance the charts with formatting and readable output.

#### Requirements
Completed program should:

- Add a chart title and axis labels
- Use colors or markers to make the plot easier to read
- Save the generated chart images to disk
