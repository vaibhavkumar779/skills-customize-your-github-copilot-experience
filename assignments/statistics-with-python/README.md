# 📘 Assignment: Statistics with Python

## 🎯 Objective

Analyze a dataset using pandas and numpy to calculate summary statistics and describe data trends.

## 📝 Tasks

### 🛠️ Load and Inspect Data

#### Description

Read the provided dataset into pandas and inspect its structure.

#### Requirements
Completed program should:

- Load data from `sample-data.csv` using pandas
- Display the first few rows of the dataset
- Show data types and missing value counts

### 🛠️ Calculate Summary Statistics

#### Description

Use pandas and numpy to compute key statistical measures for numeric columns.

#### Requirements
Completed program should:

- Calculate mean, median, standard deviation, and variance for numeric fields
- Compute the minimum and maximum values
- Return or print the statistics in a readable format

### 🛠️ Analyze Grouped Data

#### Description

Group the dataset by a category and summarize the grouped results.

#### Requirements
Completed program should:

- Group data by a categorical column such as `Category`
- Compute an aggregate statistic for each group (for example, mean sales)
- Present the grouped summary clearly
